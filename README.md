# 🏠 InteriorIQ – AI Interior Design Generator

## 📌 Overview
InteriorIQ is an AI-powered application that generates realistic interior design images for empty rooms. Users provide inputs such as room type, budget, design style, and lighting preferences, and the system generates multiple design variations using Stable Diffusion.

---

## 🚀 Features
- Upload an empty room image  
- Generate AI-based interior designs  
- Customization based on:
  - Room type (Living Room, Bedroom, Kitchen, etc.)
  - Budget (low, medium, high)
  - Style (Modern, Minimalist, Traditional)
  - Lighting (Warm, Bright, Natural, Dim)
- Generates multiple variations of designs  
- Saves output images automatically  

---

## 🛠️ Tech Stack
- Python  
- Stable Diffusion (Diffusers library)  
- PyTorch  
- PIL (Image Processing)  
- Matplotlib
